TwoNumber
---

**思路**
1. 要注意的就是list1跟list2長度可能不一樣
2. 相加後可能比9還大，需要考慮進位的情況

**解題方法**
1. 迴圈   
> * 要運用傳址呼叫的技巧，一開始讓point只給要回傳的結果，之後把下一個節點只給point，讓point前進
>  。

2. 遞迴  
> * 停止條件 l1 l2為空，而且沒有進位。
> * l1和l2相加後，將兩列node和進位結果傳入遞迴呼叫。
